# 项目简介

- **描述：** 本项目主要用于 电子产品线下门店中 管理门店的相关事宜。如 **产品、订单、工作人员、客户、店铺等**总体情况等方面信息进行集中管理。

- **技术栈：** SpringBoot、Mybatis-plus、Mysql、Vue、Nginx
